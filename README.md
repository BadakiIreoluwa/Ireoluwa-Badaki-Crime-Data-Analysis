# EDA Crime Data for Real-Estate Decision Making

This project aims to perform an exploratory data analysis of street-level crime statistics from five police forces in England and Wales: Metropolitan, North Yorkshire, Thames Valley, Devon and Cornwall, and South Wales, from February 2023 to January 2025.

## Files

- data/raw/raw_police_data: The raw unzipped crime data folder used for preprocessing and creating the crime dataset. Each Folder within this represents a month of data and contains 5 CSVs for each police force.

- data/raw/policeforceareas1991to2024.xlsx: The raw excel file containing mid-year population estimates for each age (0-85) and gender from 1991 to2024 for each police force.

- data/processed/crime_master.csv: Processed crime data

- data/processed/population_master.csv: Processed population data

- docs: Contains the Report, Statement of Work and Preprocessing Workflow. Not included in the GitHub repo

- notebooks: Contains the notebooks used in creating the cleaned data

- notebooks/eda.ipynb: Workbook used for Exploratory Data Analysis
